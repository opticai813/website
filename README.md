# Optic.ai / Embodied Labs Site

This workspace now contains a Vite + React multi-page concept site for:

- Optic.ai
- Embodied Labs

## Included

- Optic pages: home, platform, developers, government, investor, mockups
- Embodied Labs pages: home, programs, charter
- Document hub backed by local markdown files in `public/docs`
- Shared design system and animated Optic perception hero
- Printable in-app deck routes for investor, commercial, defense, and government use
- Local SVG visual assets for platform, mission-control, and frontier-lab imagery

## Run

1. Install dependencies:
   `npm install`
2. Start dev server:
   `npm run dev`
3. Build production bundle:
   `npm run build`

## Deck Export

- Open a deck route from the document hub
- Use the `Print / Save PDF` button or browser print dialog
- The deck pages include print-friendly full-slide page breaks

## Positioning Boundaries

- Optic = perception infrastructure
- Embodied Labs = frontier lab / transfer engine
- Depth = separate downstream consequence / decision infrastructure
